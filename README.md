# -ICAGnet-Incorporating-Attention-with-Deep-Learning-Approach-for-Audio-Image-Descriptions
ICAGnet is an image captioning model combining InceptionV3 for visual features with an attention-based LSTM decoder to generate accurate captions. Trained on Flickr8K and paired with a built-in TTS module, it produces both text and audio descriptions. Includes full training and inference code.
